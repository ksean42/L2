Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Будет дедлок.
На выходе из цикла в горутине канал не закрыт,
 поэтому основной поток будет заблокирован навсегда в ожидании записи в канал

```
