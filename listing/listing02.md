Что выведет программа? Объяснить вывод программы.
Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest())
}
```

Ответ:
```
Выведет 2 1. defer срабатывает на выходе из блока. 
В первой функции возращаемая переменная проинициализирована в определении функции, поэтому эта переменная хранится в области стека с возвращаемым значением.
Во второй функции Х определен внутри блока, после return Х запишется в область стека с возвращаемым значением, а defer в свою очередь обновит переменную которая хранится внутри блока.
Адрес области стека с возвращаемым значением не равен области, отведённой для внутренней переменной x.
```
